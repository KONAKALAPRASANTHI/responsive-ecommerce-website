# Responsive E-Commerce Website

A fully responsive e-commerce shopping website built using HTML, CSS, and JavaScript.

## Features
- Responsive Navbar
- Mobile Hamburger Menu
- Product Cards
- CSS Grid Layout
- Flexbox Layout
- Hover Effects
- Responsive Design using Media Queries

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Concepts Covered
- Media Queries
- Flexbox
- CSS Grid
- Responsive Web Design
- DOM Manipulation
